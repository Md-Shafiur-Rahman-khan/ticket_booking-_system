# Transport Booking System

A full-stack web application for booking bus, train, and flight tickets.

## Features
- User registration and authentication
- View available transports (bus, train, flight)
- Book seats on available transports
- View booking history
- Admin panel to manage transports
- Responsive design

## Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Server:** Apache/XAMPP/WAMP
